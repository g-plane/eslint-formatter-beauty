# eslint-formatter-beauty

Beautiful ESLint custom formatter.

## Installation

Using Yarn:

```
yarn add --dev eslint-formatter-beauty
```

Using npm:

```
npm install --save-dev eslint-formatter-beauty
```

## Usage

Run command:

```
eslint -f=beauty path/to/your/file.js
```
